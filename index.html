Data Visualisation Course Project

We are Team CFLens! Welcome to the exciting world of data visualization with CFLens! In this course project, we delve into the realms of Codeforces, a premier platform for competitive programming enthusiasts. CFLens offers a unique perspective, transcending the mere lines of code to unveil the intricate patterns, trends, and insights hidden within the vast repository of Codeforces submissions. As we embark on this journey, we aim to not only see beyond the code but also to harness the power of visualization to unravel the secrets of programming competitions. Join us as we explore the realms of data, code, and visualization to gain deeper insights into the world of competitive programming like never before.
